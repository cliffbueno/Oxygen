# Oxygen
For the manuscript on predicting the % abundance of bacterial aerobes and anaerobes in metagenomes. These were analyses done for the main publication and demonstration of the tool OxyMetaG (https://github.com/cliffbueno/oxymetag). The repository contains folders for code, data, final figures (in the manuscript), and initial figures (exploratory, not in the manuscript). The Python notebooks for the machine learning modeling were developed by Elías Stallard-Olivera, originally for another project on pathogenicity - you can ignore any text about pathogenicity.
